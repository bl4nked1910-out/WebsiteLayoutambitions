About My Ambition

Right now, I spend a lot of time online, but I don't just want to use websites and apps—I want to be the one who actually makes them. I’ve already started looking at how code works, and it’s basically like building with LEGOs, but on a computer screen. You use different languages like HTML to build the skeleton, and CSS to make it look awesome with cool colors, dark modes, and smooth animations. I want to build things that millions of people will click on. Whether it’s a massive gaming forum, an online store, or a portfolio for my friends' art, I want to write the code that brings it to life. Plus, you can do it from anywhere in the world just using a laptop, which is a pretty cool way to make a living."
